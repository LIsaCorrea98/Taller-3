Lenguajes de programación más usados

Python

Paradigma: Multiparadigma (Orientado a objetos, funcional, imperativo).

Usos: Desarrollo web, inteligencia artificial, ciencia de datos, automatización.

Ejemplo de código:

print("Hola, mundo!")

JavaScript

Paradigma: Multiparadigma (Funcional, Orientado a objetos, Imperativo).

Usos: Desarrollo web, aplicaciones móviles, servidores (Node.js).

Ejemplo de código:

console.log("Hola, mundo!");

Java

Paradigma: Orientado a objetos.

Usos: Aplicaciones empresariales, móviles (Android), servidores.

Ejemplo de código:

public class Main {
    public static void main(String[] args) {
        System.out.println("Hola, mundo!");
    }
}

C#

Paradigma: Orientado a objetos.

Usos: Desarrollo de videojuegos (Unity), aplicaciones de escritorio, desarrollo web (.NET).

Ejemplo de código:

using System;
class Program {
    static void Main() {
        Console.WriteLine("Hola, mundo!");
    }
}

SQL

Paradigma: Declarativo.

Usos: Bases de datos, análisis de datos, gestión de información.

Ejemplo de código:

SELECT * FROM usuarios WHERE edad > 18;

TypeScript

Paradigma: Orientado a objetos, funcional.

Usos: Desarrollo web, aplicaciones front-end, proyectos a gran escala.

Ejemplo de código:

let mensaje: string = "Hola, mundo!";
console.log(mensaje);

Go

Paradigma: Compilado, concurrente.

Usos: Desarrollo backend, servidores web, microservicios.

Ejemplo de código:

package main
import "fmt"
func main() {
    fmt.Println("Hola, mundo!")
}

Kotlin

Paradigma: Orientado a objetos, funcional.

Usos: Desarrollo de apps Android, aplicaciones web, backend.

Ejemplo de código:

fun main() {
    println("Hola, mundo!")
}

Rust

Paradigma: Orientado a objetos, funcional, concurrente.

Usos: Sistemas operativos, software de alto rendimiento, blockchain.

Ejemplo de código:

fn main() {
    println!("Hola, mundo!");
}

PHP

Paradigma: Multiparadigma (Imperativo, orientado a objetos).

Usos: Desarrollo web, sitios dinámicos, servidores backend.

Ejemplo de código:

<?php
echo "Hola, mundo!";
?>
